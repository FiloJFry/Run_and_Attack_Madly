# Run and Attack Madly
Run and Attack Madly è uno sparatutto in prima persona incentrato sulle battaglie boss. È un gioco molto essenziale: 0% robaccia da tossici,0% pay-to-win,100% skill,1000% tryhard. Divertiti a sconfiggere i boss più assurdi con le armi più assurde nell'ambiente più assurdo che tu abbia mai visto! 
- GIOCO NON ANCORA TERMINATO
- Ricorda di estrarre correttamente le immagini dalla cartella "Immagini.zip" prima di iniziare a giocare (quando apri la cartella "Immagini" estratta devi poter vedere subito altre cartelle)!
# Come faccio ad "estrarre correttamente" una cartella?
- Clicca sullo zip prima col tasto sinistro del mouse e poi con quello destro
- Dal menù a tendina,seleziona "estrai"
- Comparirà una cartella omonima accanto allo zip.
- Aprila e,se vedi una cartella omonima alla precedente,sposta quest'ultima nella cartella madre.
- Se il computer chiede cosa fare,seleziona "sostituisci"
